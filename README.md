# > Djinn-zsh

---

Tema e padrão de cor para uso no console zsh.

Baseado nos modelos do [Oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh).


---

As cores devem ser configuradas no próprio bash. ツ
